# Ecogram GG Personal Chatbot

A lightweight front-end chatbot template personalized for your product **Ecogram GG**.

## What it includes
- Product-themed chatbot interface.
- Profile settings (your name + chatbot focus).
- Rule-based responses for common prompts (features, pricing, strategy, support).

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

## Files
- `index.html` – chatbot layout.
- `styles.css` – styling.
- `script.js` – personalization and chatbot behavior.
